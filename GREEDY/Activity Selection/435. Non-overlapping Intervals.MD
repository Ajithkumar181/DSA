## ðŸ§  Greedy Approach (Optimal)

### âœ… Step-by-Step Strategy:

1. **Sort** all intervals by their **ending time** (i.e., `interval[1]`).
2. Initialize:

   * `end` = `intervals[0][1]` (tracks end time of last non-overlapping interval)
   * `count` = 1 (number of non-overlapping intervals we can keep)
3. **Loop** through intervals:

   * If `interval[i][0] >= end`, it's non-overlapping â†’ **keep it**, update `end`.
   * Else â†’ it's overlapping â†’ **skip it** (i.e., "remove" it)
4. Return `n - count` â†’ intervals to remove.

## âœ… Code (C++)

```cpp
class Solution {
public:
    int eraseOverlapIntervals(vector<vector<int>>& intervals) {
        if (intervals.empty()) return 0;

        sort(intervals.begin(), intervals.end(), [](vector<int>& a, vector<int>& b) {
            return a[1] < b[1];  // Sort by end time
        });

        int end = intervals[0][1];
        int count = 1;

        for (int i = 1; i < intervals.size(); i++) {
            if (intervals[i][0] >= end) {
                count++;
                end = intervals[i][1];
            }
        }

        return intervals.size() - count;
    }
};
```
