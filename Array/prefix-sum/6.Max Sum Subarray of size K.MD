## âœ… Optimal Approach: Sliding Window Technique

### ğŸ’¡ Idea:

* Compute the sum of the first `k` elements â†’ first window.
* Slide the window forward by:

  * **Adding** the next element,
  * **Removing** the element that goes out of the window.
* Track the **maximum sum** during the process.

## ğŸ§¾ C++ Code:

```cpp
int maxSumSubarrayOfSizeK(vector<int>& arr, int k) {
    int n = arr.size();
    int max_sum = 0;

    // Step 1: Calculate sum of first window
    for (int i = 0; i < k; i++) {
        max_sum += arr[i];
    }

    int window_sum = max_sum;

    // Step 2: Slide the window
    for (int i = k; i < n; i++) {
        window_sum += arr[i] - arr[i - k];
        max_sum = max(max_sum, window_sum);
    }

    return max_sum;
}
```
