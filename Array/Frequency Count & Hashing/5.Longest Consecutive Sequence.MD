## ğŸ§  Approach (Using Sorting)

### âœ¨ Idea:

1. Sort the array â†’ so that consecutive numbers are next to each other.
2. Skip duplicates.
3. Count lengths of increasing consecutive subsequences.

### âš™ï¸ Code Logic (Step-by-step)

```cpp

int count = 0;
int longest = 0;

for (int i = 0; i < n; i++) {
    if (nums[i] - 1 == lastsmall) {

    } else if (nums[i] != lastsmall) {

    }
    lastsmall = nums[i];
    longest = max(longest, count);
}
```

## âœ… Final Code (Cleaned)

```cpp
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        int lastsmall = INT_MIN;
        int longest = 0;
        int count = 0;
        int n = nums.size();

        sort(nums.begin(), nums.end());

        for (int i = 0; i < n; i++) {
            if (nums[i] - 1 == lastsmall) {
                count++;
            } else if (nums[i] != lastsmall) {
                count = 1;
            }
            lastsmall = nums[i];
            longest = max(longest, count);
        }
        return longest;
    }
};
```
