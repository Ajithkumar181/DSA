## âœ… Approach: Two Pointers / Sliding Window (O(n))

### ğŸ” Key Idea:

Use a **sliding window** by maintaining a subarray with:

* a `start` pointer and
* a `current sum`.

Shrink the window from the left when the sum becomes â‰¥ `target`.

---

### ğŸ“š Algorithm Steps:

1. Initialize:

   * `start = 0`
   * `sum = 0`
   * `minLen = INT_MAX`

2. Traverse the array using `end` pointer:

   * Add `nums[end]` to `sum`
   * While `sum â‰¥ target`:

     * Update `minLen = min(minLen, end - start + 1)`
     * Shrink the window from the left: subtract `nums[start]` and move `start++`

3. After loop:

   * If `minLen` is still `INT_MAX`, return `0`
   * Else return `minLen`

### âœ… Code (C++):

```cpp
int minSubArrayLen(int target, vector<int>& nums) {
    int n = nums.size();
    int start = 0, sum = 0, minLen = INT_MAX;

    for (int end = 0; end < n; end++) {
        sum += nums[end];

        while (sum >= target) {
            minLen = min(minLen, end - start + 1);
            sum -= nums[start++];
        }
    }

    return minLen == INT_MAX ? 0 : minLen;
}
```
